# design-portfolio

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run storybook
```

### Compiles and minifies for production
```
npm run build
```

### Push to production
```
yarn build-storybook -o ./public
git push heroku main
```
